Section 15 value functions

Gets the value from a cell based on the function used.

LEAD() LAG() FIRST_VALUE() LAST_VALUE()

LEAD & LAG

LEAD(Sales, 2, 10) OVER(PARTITION BY ProductID ORDER BY OrderDate)
the 2 is the offset (optional) is how many rows forwardfrom current row, default =1.
10 is default value, if next row is unavailable, value will be 10. Or null if not set. 

LAG(Sales, 2, 10): is the same as LEAD() except will go backwards not forward 

USE CASE:
Time Series analysis
Time Gap Analysis

FIRST_VALUE() LAST_VALUE() Notes:
LAST_VALUE() default settings do not work well. Bounds need to be specified. 

LAST_VALUE(Sales) OVER(ORDER BY Month ROWS BETWEEN CURRENT ROW AND UNBOUNDED FOLLOWING)
